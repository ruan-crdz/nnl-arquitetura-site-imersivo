# nnl-arquitetura-site-imersivo
Website institucional imersivo da NNL Construtora &amp; Engenharia, desenvolvido com HTML, CSS e JavaScript puros. O projeto propõe uma experiência futurista inspirada em arquitetura digital, com carrossel 3D, seção imersiva, FAQ interativo e identidade visual baseada em azul profundo e dourado metálico.
